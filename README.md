# babel-typescript-react-webpack-example
Practical configuration example for Babel 7 + React + Webpack

## Contains
- [x] TypeScript 3
- [x] Babel 7
- [x] Webpack 4
- [x] Jest
- [x] TSLint

## Set up
Following commands install dependencies and start dev server:
```
yarn
yarn run start
```
